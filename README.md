# 📈 BourseInfo — Actualités Bourse de Casablanca

Un site web moderne et responsive dédié aux actualités de la **Bourse de Casablanca (MASI)**.

---

## 🚀 Demo

> Disponible sur GitHub Pages : `https://yourusername.github.io/bourse-casablanca`

---

## ✨ Fonctionnalités

- 📡 Ticker en temps réel (MASI, MADEX, actions)
- 📊 Statistiques clés du marché (MASI, MADEX, Volume, Sociétés cotées)
- ⭐ Article à la une mis en avant
- 📰 Liste des dernières actualités avec filtres par catégorie
- 📈 Mini-graphique hebdomadaire du MASI
- 🏆 Meilleures performances du jour
- 📬 Formulaire d'inscription à la newsletter
- 📱 Design responsive mobile-friendly

## 🌍 Déploiement sur GitHub Pages

1. Aller dans **Settings** → **Pages**
2. Source : **Deploy from a branch**
3. Branch : `main` / `root`
4. Cliquer **Save**
5. Votre site sera en ligne en quelques secondes ✅
